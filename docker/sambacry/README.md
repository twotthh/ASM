# 🐳 SambaCry (CVE-2017-7494) 실습 환경

이 폴더는 교수님 PPT 가이드라인에 맞춘 **SambaCry (CVE-2017-7494) 취약점**의 Docker 서버 구축, Nmap/Nuclei 탐지 실습 및 조치 방안을 정리한 공간입니다.

## 🗂️ 포함된 파일 안내
* **[final_report.md](./final_report.md)**: 1단계(구축), 2단계(탐지 및 스캔 결과), 3단계(관리자 조치 가이드)가 상세히 기록된 최종 보고서 파일입니다.
* **실행 영상**: 취약점 탐지 성공 과정(Nuclei 마킹 결과 포함)을 담은 전체 구동 영상은 팀 단톡방에 업로드 완료했습니다.
